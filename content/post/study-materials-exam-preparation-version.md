## 考綱：

1. **排隊論模型**：学習簡單的排隊論模型的建模方法，熟悉和掌握排隊系统中的平均隊长、平均顧客數、系统中顧客退留時间、隊列中顧客等待時间等的計算。
2. **規划論模型**：学習掌握數学規划：線性規划、整數規划、目標規划、动態規划等相關概念和理論，并且要求能用相關的方法建立對應的數学規划模型和求解。
3. **插值与拟合建模**：掌握曲線拟合、插值的基本方法，学会用插值、拟合作數据处理。
4. **回歸分析模型**：学習掌握線性回歸与非線性回歸的相關概念和理論，并且要求能用相關的最小二乘方法建立對應的回歸分析模型，井能進行相關的概率统計進行檢驗。
5. **差分方程模型**：了解差分法的基本思想，学会建立实际問題的离散模型，掌握递推、迭代法等求解過程，理解和掌握蛛网模型、银行貨款等問題。
6. **常微分方程模型**：熟悉微分方程建模的基本步驟，掌握線性微分方程建模基本方法，了解非線性微分方程模型的一些特殊性質：熟悉微分方程的解法。
7. **决策論模型**：掌握決策論的基本理論及其應用。
8. **圖和网絡模型**：掌握最短路、最小生成樹、最大流等的原理与求解。

### 四、回歸分析模型

- 回歸：是一种统計分析方法，用于預測一个連續变量与一个或多个其他变量之间的關系。
- 線性回歸：是回歸分析的一种，假设因变量与自变量之间存在線性關系，即因变量与自变量的關系可以用一条直線來描述。
- 非線性回歸：是回歸分析的一种，假设因变量与自变量之间不存在線性關系，即因变量与自变量的關系不能用一条直線來描述，需要用更复雜的函數模型來描述。

  一元線性回歸模型：$ y = \beta_0 + \beta_1 x$ <br>

  $$\color{red}\begin{cases} 斜率 \beta_1：\frac{\sum_{i=1}^n x_i y_i - n \bar{x} \bar{y}}{\sum_{i=1}^n x_i^2 - n \bar{x}^2} \\\ 截距 \beta_0：\bar{y} - \beta_1 \bar{x}\end{cases}$$

  檢驗方法：課本是**假设檢驗**判斷是否爲線性，真題沒有考

### 五、差分方程

差分:$\Delta y_{x} = y_{x+1} - y_{x}$

二階差分：$\Delta {\Delta y_{x}} = \Delta^2 y_{x} = y_{x+2} - 2y_{x+1} +y_{x}$

一階線性差分方程的一般形式爲：$ y_{x+1} = ay_x + f(x)$

1. **齐次方程：**
   如果 $f(x) = 0$，方程变爲齐次方程：$ y_{x+1} = ay_x $ <br>
   其解爲：<br>
   $$ \color{red}{y_x = C a^x} $$
   
2. **非齐次方程：**

   - 如果 $f(x) = b$：

   递推得其解爲：

   $$ \color{red}{f(n) = \begin{cases} bx+c & \text{, } a = 1 \\\ \frac{b}{1-a}+C\cdot a^x & \text{, } a \neq 1 \end{cases}}$$

   <br>

   - 如果 $f(x) = (a_0 + a_1x + \cdots + a_nx^n)b^x$
   
   其特解爲(記得還要加通解也就是對應齐次方程的解)：
   
   $$\color{red}{y_x^* = x^k(A_0 + A_1x + \cdots + A_nx^n) \cdot b^x \\\ 其中 A 根据 f(x)多項式系數判斷，k = \begin{cases} 0 & \text{, } a \neq b \\\  1 & \text{, } a =b \end{cases}}$$

例題：某城市 2020 年汽车保有量 32 万輛，預計此后每年報廢上一年的汽车保有量的 6%，并且每年新增汽车數量相同。爲保护城市環境，該城市計划到 2035 年汽车保有量不超過 50 万輛，那么每年新增汽车數量不應超過多少輛？（$（0.94)^{15} = 0.395$）

解：设第 n 年的汽车保有量爲$a_n$，每年新增汽车數量爲 x。

1. 由題：$a_n = 0.94a_{n-1} + x$

2. 其通解爲：
   $a_n = C(0.94)^n + \frac{x}{1-0.94}$
   其中 C 是常數，需要根据初始条件确定

3. 利用初始条件 $a_0 = 32$，我們可以求出 C：
   $32 = C + \frac{x}{0.06}$
   $C = 32 - \frac{x}{0.06}$

4. 即：
   $a_n = (32 - \frac{x}{0.06})(0.94)^n + \frac{x}{0.06}$

5. 現在，需要滿足 15 年后不超過 50 万輛的条件：
   $a_{15} ≤ 50$

6. 代入 n=15 到剛剛的解：
   $(32 - \frac{x}{0.06})(0.94)^{15} + \frac{x}{0.06} ≤ 50$

   得：$x≤ 3.71$万

### 七、决策論模型

一般会根据樂觀准則、悲觀准則、折衷主义准則進行决策

過程其实很簡單：

- 先看列，樂觀准則找每列最大值，悲觀准則找每列最小值，折衷主义准則算出$（每列最大值）\cdot  \lambda +(每列最小值) \cdot (1-\lambda)$
- 找到后取最大值

例題: 某公司將推出一种新產品，有三种推广方案：讓利銷售 $(d_1)$，送貨上门 $(d_2)$，不采取措施$(d_3)$。未來市場可能有暢銷 $s_1$、一般$s_2$、滯銷 $s_3$三种狀態。假设事先不知道這三种自然狀態的出現的概率，但知道各种狀態下各方案的盈利，见表。請用以上介紹的 3 种决策方法進行决策（樂觀系數 $\alpha = 0.6$）。

|      市場狀態      | 暢銷 $s_1$ | 一般  $s_2$ | 滯銷 $s_3$ |
| :----------------: | :--------: | :---------: | :--------: |
|        方案        |    盈利    |    盈利     |    盈利    |
|  讓利銷售 $(d_1)$  |     60     |     10      |     -6     |
|  送貨上门 $(d_2)$  |     30     |     25      |     0      |
| 不采取措施 $(d_3)$ |     10     |     10      |     10     |

解：1、**樂觀法**:$ \underset{i}{max}\text{ }\underset{j}{max}{a_{ij}} $

$max \{60, 10, -6\} = 60 $<br>
$max \{30, 25, 0\} = 30$<br>
$max \{10, 10, 10\} = 10 $<br>

$ \underset{i}{max}\text{ }\underset{j}{max}{a_{ij}} =60 $

對應的最优方案爲 $(d_1)$。

2. **悲觀法**:$ \underset{i}{max}\text{ }\underset{j}{min}{a_{ij}} $

$min \{60, 10, -6\} = -6 $<br>
$min \{30, 25, 0\} = 0 $<br>
$min \{10, 10, 10\} = 10 $<br>

$ \underset{i}{max}\text{ }\underset{j}{min}{a_{ij}} =10$

對應的最优方案爲 $(d_3)$。

3. **樂觀系數法** $\alpha = 0.6$:

$CV_1 = 0.6 \times 60 + 0.4 \times (-6) = 33.6 $<br>
$CV_2 = 0.6 \times 30 + 0.4 \times 0 = 18 $<br>
$CV_3 = 0.6 \times 10 + 0.4 \times 10 = 10 $<br>
$\max \{33.6, 18, 10\} = 33.6 $

故最优方案爲 $(d_1)$。

### 八、圖和网絡模型

###### 最短路徑問題

最短路徑是把两点之间路徑最短的問題，它只着眼于点与点之间的路徑問題，并不關注整个圖，也就意味着對一个節点运行算法的結果与另一个節点的結果之间沒有多少關系。

**Dijkstra 算法**：

- 步驟

  1. 初始化：设置起始頂点的距离爲 0，其他所有頂点的距离爲无限大。起始頂点加入未处理頂点集合。
  2. 選擇未处理頂点集合中距离起始頂点最近的頂点作爲當前頂点。
  3. 更新當前頂点的邻居頂点的距离（如果通過當前頂点到达邻居頂点的路徑更短，則更新邻居頂点的距离）。
  4. 將當前頂点標記爲已处理，并从未处理頂点集合中移除。
  5. 重复步驟 2-4，直到处理完所有頂点或目標頂点的距离不再更新。

  

###### 最小生成樹問題

最小生成樹是把連通的圖的**所有頂点**連起來路徑之和最小的問題，即生成樹总權值之和最小。即**在一个連通的圖里，如何去除圖里的边，使得剩余的边仍能連接所有的節点，且這些边的權重之和最小**。

**Kruskal 算法**：

- 步驟
  1. 初始化一个空的生成樹。
  2. 依次選擇權重最小的边，如果這条边加入生成樹后不形成環，則將其加入生成樹。
  3. 重复步驟 2，直到生成樹包含圖中的所有頂点。
