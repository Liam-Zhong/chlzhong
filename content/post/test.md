## 什麼是私有雲運算？

私有雲端運算是蘋果公司基於伺服器的 Apple Intelligence 擴充服務。當某個任務需要的功能超出設備端模型所能提供的範圍時，它會在蘋果自研晶片伺服器上運行，並提供與用戶期望從設備端獲得的相同隱私保障 — — 請求不會被存儲，不會用於訓練，並且可以由獨立的安全研究人員進行驗證。

對於開發者而言，PCC 模型相比設備端基礎模型提供了有意義的升級：

- **32K 個令牌的上下文窗口**，相較之下，裝置上的上下文視窗為 4K。
- **更強大的推理能力**，推理等級可配置
- **無需身份驗證或 API 金鑰管理** — — 隱私和身分由系統處理。
- 符合條件的開發者**無需支付任何雲端 API 費用**

## 哪些人符合資格？

符合以下所有條件的開發者可以存取 PCC：

1. **已加入 App Store 小型企業計劃**
2. 所有應用程式在**App Store 的首次下載量不到 200 萬次**
3. 分配給您的開發者帳戶的**私有雲運算授權**

在 Apple Intelligence 可用的情況下，符合資格的開發者可以在 App Store 上發布的 App 中使用 PCC，並透過 TestFlight…
