windows 專业版和家庭版有哪些區別？

## 

![](https://picx.zhimg.com/50/v2-218d1faa42206306d3fedef802d84ac7_720w.jpg?source=2c26e567)

## 前言

這个文章所有內容來源于微軟官网，爲个人备份使用，更具体的差別可以在各类网站搜索。

普通消費者能接触到的视窗系统分爲標題所述三个版本，本文不闡述所有版本的共性，只說缺少的功能，如果您覺得需要用到，可自行購買升級。

其中家庭版售价 $139 或 ¥1088

專业版售价 $199，專业工作站版 $309，微軟官网未給出人民幣价格。


---


## 家庭版缺少功能包括

1. **BitLocker 设备加密** 
2. 集成 Microsoft 信息保护功能
3. 遠程部署和合規性
4. Windows Hello 企业版
5. Windows 信息保护 (WIP)
6. **分配的訪問權限** 
7. Azure Active Directory 联接
8. 混合 Azure Active Directory 联接
9. 云管理更新
10. 使用 Azure 的企业狀態漫游
11. eSIM 企业解决方案
12. **組策略** 
13. 展台模式设置
14. 移动應用管理
15. 移动设备管理
16. 組织信息
17. 遠程協助
18. 本地應用和虛拟應用无縫銜接
19. Active Directory 支持
20. Azure Active Directory 支持
21. Windows Autopilot
22. Windows Server Update Service (WSUS)  
23. 适用于企业的 Windows 更新
24. **遠程桌面** 

粗体功能是我个人覺得普通用户可能会用到的功能。另外官网未提及，实际家庭版缺乏的功能還有 **Hyper-V** 

## 專业版缺少功能包括

1. REFS
2. Persistent memory
3. SMB Direct

看上去都是不怎么需要的功能。
