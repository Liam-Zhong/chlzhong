## 前言

这个文章所有内容来源于微软官网，为个人备份使用，更具体的差别可以在各类网站搜索。

普通消费者能接触到的视窗系统分为标题所述三个版本，本文不阐述所有版本的共性，只说缺少的功能，如果您觉得需要用到，可自行购买升级。

其中家庭版售价 $139 或 ¥1088

专业版售价 $199，专业工作站版 $309，微软官网未给出人民币价格。


---


## 家庭版缺少功能包括

1. **BitLocker 设备加密** 
2. 集成 Microsoft 信息保护功能
3. 远程部署和合规性
4. Windows Hello 企业版
5. Windows 信息保护 (WIP)
6. **分配的访问权限** 
7. Azure Active Directory 联接
8. 混合 Azure Active Directory 联接
9. 云管理更新
10. 使用 Azure 的企业状态漫游
11. eSIM 企业解决方案
12. **组策略** 
13. 展台模式设置
14. 移动应用管理
15. 移动设备管理
16. 组织信息
17. 远程协助
18. 本地应用和虚拟应用无缝衔接
19. Active Directory 支持
20. Azure Active Directory 支持
21. Windows Autopilot
22. Windows Server Update Service (WSUS)  
23. 适用于企业的 Windows 更新
24. **远程桌面** 

粗体功能是我个人觉得普通用户可能会用到的功能。另外官网未提及，实际家庭版缺乏的功能还有 **Hyper-V** 

## 专业版缺少功能包括

1. REFS
2. Persistent memory
3. SMB Direct

看上去都是不怎么需要的功能。
